** Set postgresql authentication

   #+BEGIN_SRC shell
   vim /etc/postgresql/10/main/pg_hba.conf
   #+END_SRC

   change the line of
   'local   all             all                                  peer' into
   'local   all             all                                  md5'

   restart service:

   #+BEGIN_SRC shell
   sudo service postgresql restart
   #+END_SRC
